Python’dagi Belgi To‘plamlari va Kodlash (Encoding/Decoding)

1. Python 2 va Python 3: Belgilarni Saqlashdagi Farq

Python 2

  • Satrlar ASCII formatida saqlangan: Python 2 matnli ma’lumotlarni 8-bit ASCII sifatida ifodalagan.
  • Kodlash cheklangan: ASCII faqat 127 ta belgini o‘z ichiga olgan.
  • Xalqaro tillar uchun qo‘shimcha kodlashlar ishlatilgan: Masalan, Latin-1, Windows-1252, Shift_JIS.

Python 3

  • Unicode ichki format sifatida qabul qilingan.
  • Dunyo tillarining hammasini qo‘llab-quvvatlaydi.
  • Unicode muammolarni hal qildi va bugungi kunda standartga aylangan.

2. Unicode va UTF-8 O‘rtasidagi Farq

Unicode xalqaro standart bo‘lib, har bir belgi o‘ziga xos kodga ega. Uni turli formatlarda saqlash mumkin:

  • UTF-8: Eng ko‘p ishlatiladigan format, ASCII bilan mos.
  • UTF-16: 2 yoki 4 bayt, Windows va Java tizimlarida ishlatiladi.
  • UTF-32: Har bir belgi 4 bayt, juda katta hajm talab qiladi.

3. Kodlash (Encoding) va Dekodlash (Decoding)

Kodlash:

text = "Salom"
encoded_text = text.encode('utf-8')
print(encoded_text)  # b'Salom'

Dekodlash:

decoded_text = encoded_text.decode('utf-8')
print(decoded_text)  # Salom

4. Python 3 da Fayllar, Tarmoqlar va Ma’lumotlar Bazalarida Kodlash

Fayllar bilan ishlash:

with open("file.txt", "w", encoding="utf-8") as f:
    f.write("Salom Dunyo")

Tarmoq orqali kodlash:

import socket
s = socket.socket()
s.connect(('example.com', 80))
s.send(b'GET / HTTP/1.1\r\nHost: example.com\r\n\r\n')
response = s.recv(4096)
print(response.decode('utf-8'))

Ma’lumotlar bazasi bilan ishlash:

import sqlite3
conn = sqlite3.connect("database.db")
conn.execute("PRAGMA encoding = 'UTF-8'")

5. Xulosa

  • Python 3 Unicode ishlatadi, lekin tashqi tizimlar UTF-8 talab qiladi.
  • Agar belgilar noto‘g‘ri chiqsa, kodlash va dekodlashni tekshirish kerak.
